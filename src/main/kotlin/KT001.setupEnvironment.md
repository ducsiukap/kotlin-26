# Kotlin environment setup.

> To setup environment for Kotlin program, have to install `jdk` and `kotlin-compiler`.

## Index

1. [Install JDK](#1-install-jdk-)
2. [Install Kotlin (kotinc)](#2-download-the-kotlin-compiler)

## 1. Install JDK

Following steps to install `JDK`:

- Step 1: Download Oracle Java Development Kit - JDK:  
  Link: **[Download JDK - Java Deverlopment Kit (Oracle)](https://www.oracle.com/java/technologies/downloads/)**  
  Note:
    - Select the JDK version that matches with OS.
    - Download LTS (Long-Term Support) version.
- Step 2: Open the installed file and install.
- Step 3: Setup local Java environment:
    + 3.1. Go to **C:\Program Files\Java\jdk-25\bin** and copy this directory path.
    + 3.2. Open search, type "View advanced system setting" and open.
    + 3.3. Click **Enviroment Variables..**, go to **System variables**.
    + 3.4. Click new, set:
      ```text
      Variable name = JAVA_HOME
      Variable value= <the copied path in 3.1 (remove "/bin" at the end)>
      ```
      then click **OK**.
    + 3.5. Select **PATH** and click **Edit**.
    + 3.6. Click **New**, paste _**the copied path**_ or _**"%JAVA_HOME%/bin"**_ to the input box, then click **OK**.
    + 3.7. Finally, just click **OK** > **Apply** > **OK**.
- Step 4: Verify the installation
  Open `cmd` (Command promt) and run this command:
    ```bash
    java --version
    ```
  The result should belike:
    ```bash
    java 21.0.8 2025-07-15 LTS
    Java(TM) SE Runtime Environment (build 21.0.8+12-LTS-250)
    Java HotSpot(TM) 64-Bit Server VM (build 21.0.8+12-LTS-250, mixed mode, sharing)
    ```
  If no, the installation is failed.

## 2. Download the Kotlin compiler

Following steps:

- Step 1: Download `kotlin-compiler`:  
  Link: **[Github Release](https://github.com/JetBrains/kotlin/releases/tag/v2.3.0)**  
  Go to the ultimate bottom of the website, then you will see Assets. In that, download
  _**`kotlin-compiler-x.x.x.zip`**_
- Step 2: Setup the Kotlin compiler for the command line
    + 2.1. Extract the downloaded file.
    + 2.2. Open the extracted, go to `bin` directory and copy its path.
    + 2.3. Open `View advanced system settings` > `Environment variables`.
    + 2.4. On "System variables", click `edit`.
    + 2.5. Click `new` and paste the copied path.
    + 2.6. Click `ok` > `ok` > `ok`.
- Step 3: Verify the installation
    - Check the version:
      Open `Command Prompt` and run this command:
      ```bash
      kotlin -version
      ```
      The output should belike:
      ```bash
      C:\Users\admin>kotlin -version
      Kotlin version 2.2.0-release-294 (JRE 25+37-LTS-3491)
      ```
    - Or run this command:
        ```bash
        kotlinc -Xrepl
        ```
      The result should belike:
        ```bash
        C:\Users\admin>kotlinc -Xrepl
        WARNING: A terminally deprecated method in sun.misc.Unsafe has been called
        WARNING: sun.misc.Unsafe::objectFieldOffset has been called by com.intellij.util.containers.Unsafe
        WARNING: Please consider reporting this to the maintainers of class com.intellij.util.containers.Unsafe
        WARNING: sun.misc.Unsafe::objectFieldOffset will be removed in a future release
        Welcome to Kotlin version 2.2.0 (JRE 25+37-LTS-3491)
        Warning: this REPL implementation is deprecated and will be removed soon.
        Warning: REPL is not compatible with the Kotlin version 2.2.0, using '-language-version 1.9'.
        Type :help for help, :quit for quit
        >>>
        ```
